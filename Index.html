Guía Paso a Paso para Completar y Escalar tu Negocio

Aquí encontrarás todos los pasos detallados para transformar tu landing en una plataforma robusta y escalable. Haz clic en cada ítem para desplegar la información completa.

1. Definir y recopilar contenidos y marca
  • Identidad de marca:
    • Paleta de colores: define 2–3 colores principales (p. ej. púrpura místico, dorado, blanco) y colores de acento.
    • Tipografía: Burgundia para títulos, Poppins para textos. Asegura tener el archivo burgundia.ttf y definiciones de tamaños/pesos.
    • Logotipo: en PNG/SVG para nav-bar y favicon.
    • Imágenes:
      • nayra.png para Hero.
      • Iconos/imágenes para servicios: tarot.png, cuarzos.png, rituales.png, etc.
      • Favicon: favicon.ico o favicon.png.
  • Textos finales:
    • Titulares y subtítulos de cada sección.
    • Descripciones detalladas de cada servicio (tarot, cuarzos, rituales...).
    • “Sobre mí” o biografía breve.
    • Llamadas a la acción (CTA): “Reserva tu lectura”, “Contáctame”, etc.
    • Políticas: política de privacidad, términos de uso adaptados.
  • Estructura de páginas:
    • Landing: Hero, presentación, servicios destacados, testimonios (opcional), contacto.
    • Páginas secundarias opcionales: servicios individuales (tarot.html, cuarzos.html), sobre mí extendido, blog, testimonios, tienda, página de agradecimiento, política de privacidad, términos y condiciones.
  • Recopilar medios y optimizar:
    • Optimiza imágenes (TinyPNG, Squoosh).
    • Imágenes responsive: versiones pequeñas para móviles.
    • Estructura de carpetas: 
      /nayra-elhara/
├── index.html
├── gracias.html
├── style.css
├── favicon.png
├── images/
│   ├── nayra.png

│   └── ...
└── fonts/
    └── burgundia.ttf

  .nav-links a { font-size: 1rem; }
  .services { flex-direction: column; gap: 1.5rem; }
  .service-card { max-width: 90%; margin: 0 auto; }

  • Menú hamburguesa:
    • Agrega un ícono “☰” visible <768px para togglear la visibilidad de ul.nav-links.

3. Contacto funcional: configurar envío de formulario
  • Formulario con Formspree:
  • JavaScript para enviar con fetch y mostrar mensaje inline: 
    document.addEventListener('DOMContentLoaded', function() {
  const form = document.getElementById('contactForm');
  if (!form) return;
  form.addEventListener('submit', function(e) {
    e.preventDefault();
    const btn = form.querySelector('button[type="submit"]');
    if (btn) {
      btn.disabled = true;
      btn.innerText = 'Enviando...';
    }
    fetch(form.action, {
      method: 'POST',
      body: new FormData(form),
      headers: { 'Accept': 'application/json' }
    })
    .then(response => {
      if (response.ok) {
        const container = document.createElement('div');
        container.className = 'form-success';
        container.innerText = '¡Mensaje enviado! Gracias por conectar. Te responderé pronto.';
        form.replaceWith(container);
      } else {
        alert('Hubo un error al enviar tu mensaje. Por favor inténtalo nuevamente o recarga la página.');
        if (btn) {
          btn.disabled = false;
          btn.innerText = 'Enviar mensaje';
        }
      }
    })
    .catch(() => {
      alert('Error de red. Intenta de nuevo en unos momentos.');
      if (btn) {
        btn.disabled = false;
        btn.innerText = 'Enviar mensaje';
      }
    });
  });
});
  • Página de agradecimiento: crea `gracias.html` en la raíz con rutas corregidas.

6. SEO básico y meta tags
  • En <head>: <title>, , Open Graph, Twitter Card, favicon.
  • Sube imagen OG a `https://nayraelhara.com/images/og-image.jpg`.
  • robots.txt y sitemap.xml en tu hosting: `https://nayraelhara.com/sitemap.xml`.
7. Analítica y seguimiento
  • Google Analytics 4: inserta gtag.js con tu ID en o antes de .
  • Search Console: verifica `https://nayraelhara.com` y envía sitemap.
  • Eventos personalizados: medir clics en CTA, envíos de formulario.
8. Hosting y despliegue
  • Hosting estático: Netlify, Vercel, GitHub Pages. Apunta DNS de `nayraelhara.com` según instrucciones.
  • Hosting con backend: si necesitas funciones serverless o API propias.
  • SSL automático tras apuntar dominio.
9. Dominio y SSL
  • Comprar dominio (nayraelhara.com) y apuntar a hosting.
  • Configurar DNS: registros A/CNAME según proveedor de hosting.
  • SSL/TLS configurado automáticamente.
10. Optimización de rendimiento
  • Imágenes optimizadas, WebP si es posible.
  • Minificar CSS/JS en producción.
  • Preload de fuentes (burgundia.ttf o WOFF2).
  • Lazy loading en imágenes no críticas (loading="lazy").
  • Lighthouse para auditar performance y accesibilidad.
11. Integración de programación de sesiones
  • Calendly: incrusta widget o enlace popup con tu URL de Calendly.
  • Sincronizar Calendar para evitar doble reserva.
  • Alternativa: enlace a WhatsApp Business.
12. E-commerce ligero para kits energéticos
  • Shopify / WooCommerce o Buy Button para vender kits de cuarzos/rituales.
  • Pagos: Stripe/PayPal.
  • Automatización con Zapier: confirmaciones y seguimientos.
  • Mostrar productos en sección “Tienda” o landing.
13. Captación de leads y email marketing
  • Formulario de suscripción (Mailchimp/Sendinblue) con lead magnet.
  • Secuencia de bienvenida automatizada.
  • Newsletter periódica con reflexiones, testimonios y ofertas.
14. Analytics avanzado y seguimiento de conversiones
  • Eventos en GA4 para medir conversiones (clic en “Reserva tu lectura”, envíos de formulario).
  • Meta Pixel para remarketing en campañas pagadas.
  • Páginas de “Gracias” para medir conversiones en Ads.
15. Seguridad y confianza
  • HTTPS (SSL) configurado.
  • Páginas de Política de privacidad y Términos de uso.
  • Sellos de pago seguro (PayPal, Stripe) si se vende.
  • Actualizar dependencias/plugins si usas CMS.
16. Pruebas y feedback
  • Pruebas de usuario: pedir feedback sobre UI/UX.
  • Corregir legibilidad y compatibilidad de formularios.
  • Monitoreo de errores (404, scripts que no cargan).
17. Lanzamiento y marketing inicial
  • Redes Sociales: anunciar lanzamiento con posts y videos.
  • Colaboraciones con influencers o aliados en espiritualidad/tecnología.
  • Eventos en vivo: webinars o sesiones gratuitas.
  • Publicidad pagada (Meta Ads, Google Ads) segmentada.
  • SEO orgánico: blog con artículos optimizados.
18. Iteración y escalabilidad
  • Analizar métricas: servicios más demandados, fuentes de tráfico.
  • A/B testing: titulares, colores, CTA.
  • Nuevos servicios/productos: cursos, membresías premium.
  • Automatización con IA: chatbot espiritual con tu estilo.
  • Plataforma multi-terapeuta: invitar a otros sanadores.
  • Apps móviles: diario emocional o meditaciones.
  • Experiencias inmersivas: VR/AR de rituales.
  • Internacionalización y pagos internacionales.
  • Inversión/partnerships con plataformas de bienestar digital.
19. Gestión y operaciones
  • Herramientas de productividad: Notion/Trello/Asana.
  • CRM ligero: gestionar clientes y citas (Calendly + Google Calendar o CRM).
  • Automatización de emails: recordatorios, encuestas, upsell de kits.
  • Contabilidad y facturación según región.
  • Soporte al cliente: email, WhatsApp Business, chat en vivo.
  • Equipo: contratar o colaborar según crecimiento.
20. Roadmap y visión a largo plazo
  • Definir hitos: MVP, marketing, IA, app móvil.
  • KPIs: conversiones, retención, ingresos recurrentes.
  • Validación continua: encuestas, feedback, reviews.
  • Reinversión o buscar inversores.
  • Innovación constante: explorar nuevas tecnologías.
